Automatic Waste Sorter

Collaborated with a team of four to develop the Automatic Waste Sorter, a machine learning-based waste management system that sorts waste into recyclable, organic, and non-recyclable categories. CD Spotify Jukebox

Designed and custom-built a CD Spotify Jukebox utilizing a Raspberry Pi Zero and servo motors to control a robotic arm for picking up and displaying CDs. Developed using Python software, including album playback through the Spotify API. Utilized CAD software to design and 3D print all parts, and implemented illuminated buttons and dynamic lighting. Demonstrated strong problem-solving and technical skills through assembly, fine-tuning, and testing of the robotic arm.

Basketball Scoreboard

As part of a team of three, designed and implemented a basketball scoreboard on an FPGA board using Verilog. The scoreboard features a timer and two separate point displays on one 7-segment display. Implemented user input for stopping the timer and adding 1, 2, and 3 points. Contributed to the design and coding of the scoreboard logic and display components, demonstrating proficiency in Verilog and digital design.
